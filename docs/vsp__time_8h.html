double vsp_time_cpu_double ( void  )

Get the amount of CPU time used by the current process in seconds.

Returns -1.0 if an error occurred. Time is measured since an arbitrary and OS-dependent start time. The returned real time is only useful for computing an elapsed time between two calls to this function.
