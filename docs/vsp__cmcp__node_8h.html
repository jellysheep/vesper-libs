vsp_cmcp_node* vsp_cmcp_node_create ( vsp_cmcp_node_type  node_type)

Create new vsp_cmcp_node object.

The state of the newly created node is set to VSP_CMCP_NODE_UNINITIALIZED. Returned pointer should be freed with vsp_cmcp_node_free(). Returns NULL and sets vsp_error_num() if failed.

void **  message_buffer 
)

Wait for and receive message from the node's subscribe socket.

Blocks until message could be received. Stores a pointer to the message data and returns the data length. The message data has to be freed with nn_freemsg(). Returns negative value and sets vsp_error_num() if failed.
