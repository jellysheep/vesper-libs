Create new vsp_cmcp_message object.

Returned pointer should be freed with vsp_cmcp_message_free(). The vsp_cmcp_message will not free the vsp_cmcp_datalist object and it has to be accessible until vsp_cmcp_message_free() is called. Returns NULL and sets vsp_error_num() if failed.
